# Screenshots

Drop the following PNGs here to populate the README screenshots section:

- `home.png` — Home dashboard view (connection status, pending counts, activity timeline)
- `review.png` — Plan Review canvas with annotations and multi-session tabs
- `question.png` — Structured Questions canvas

Recommended dimensions: 1600–1920 px wide, retina-quality. PNG with transparent / matching theme background works best.

These files are referenced from the top-level [`README.md`](../../README.md). Until they exist, GitHub will render broken-image placeholders in their slots — this is intentional, marking where to drop captures.
